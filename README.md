# 🌲 Deforestation

> Proyecto desarrollado en Unity como práctica de desarrollo de videojuegos.

---

## 📖 Descripción

Deforestation es un videojuego en 3D desarrollado con Unity en el que el jugador debe explorar un entorno natural, recolectar recursos y utilizar una máquina especializada para cumplir su objetivo antes de que el entorno se vuelva demasiado peligroso.

El proyecto combina exploración, gestión de recursos, interacción con el escenario y conducción de una máquina capaz de modificar el terreno.

---

## 🎮 Características

- Exploración en primera persona.
- Recolección de diferentes recursos.
- Sistema de inventario.
- Control de una máquina independiente del jugador.
- Sistema de vida.
- Enemigos con IA mediante NavMesh.
- Interfaz de usuario.
- Menú de configuración.
- Sistema de victoria.
- Sistema de derrota.
- Sistema de interacción con objetos.

---

## 🎯 Objetivo

El jugador debe:

1. Explorar el mapa.
2. Recolectar recursos.
3. Utilizar la máquina para desplazarse y trabajar.
4. Entregar los recursos necesarios.
5. Completar la misión antes de quedarse sin vida o recursos.
